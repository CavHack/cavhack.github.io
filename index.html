Welcome to CavHack's DALi Manifesto.

With the advent of today's 'NoStack-Information-Universe', Modern A.I Libraries require to scrape, parse, encode, and infer intent, from Sparse and sometimes un-related information (i.e: geolocation coordinates, time of activity, session duration, and even text-message HSL). DALi is built with Clojure/Smalltalk and aims to integrate all of Today's most in-vogue Machine Learning libraries, into a functional, portable, and context-agnostic probabilistic NLP. What makes us different from others is our passion for Graph Topology and Abstract mathematics, the binding of the collective unconscious and the human condition. DALi is fairly nascent, but expanding stochastically.

About DALi

DALi stands for Directed-Acyclic-Linked-Intelligence and provides an efficient Open-Source API written with the functional programmer in mind. DALi utilizes Clojure which exposes our algorithms to the Java Virtual Environment. There we can communicate with clusterized agents via Apache Storm. Clojure is inspired by the Torch library, which has become the staple of today's Machine Learning archetypes. By focusing on KISS principles, DALi is powerful enough to distribute tasks a long a variety of containers; employing today's best-practices with Docker.
